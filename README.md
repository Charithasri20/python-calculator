# Simple Calculator — Python Project

## Overview
A command-line calculator that performs addition,
subtraction, multiplication, and division with
complete error handling.

## Features
- Supports +, -, *, / operations
- Handles division by zero gracefully
- Handles invalid number input
- Runs continuously until user types quit

## Technologies Used
- Python 3

## How to Run
1. Open notebook in Google Colab
2. Click Runtime → Run all
3. Enter numbers and operator when prompted

## Concepts Demonstrated
- Functions with parameters and return values
- While loops for continuous program flow
- try/except error handling
- Input validation
- if/elif/else conditions
